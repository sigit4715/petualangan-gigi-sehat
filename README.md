# 🦷 Petualangan Gigi Sehat

Interactive dental health education website for Indonesian children.

## Features
- 10 scene scrollytelling
- Boss battle vs Cavitarus (1000 HP)
- Sound effects (Web Audio API)
- Floating particles + star rating
- Mini game brush teeth (30s timer)
- Quiz (5 questions)
- Responsive mobile + desktop

## How to use
Open `index.html` in any browser. Zero dependencies!

## Tech
Pure HTML5 + CSS3 + JavaScript ES6. No frameworks.
